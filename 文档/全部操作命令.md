以下是完整的可操作指令清单及说明。所有命令均通过 src/cli/claw_interface.py 执行。

## 一、数据采集指令

### 1. 更新主板股票列表

bash复制下载

```text
python src/cli/claw_interface.py update_stock_list
```- 从 AKShare 获取实时行情，生成股票列表。
- 输出文件：data/processed/metadata/stock_list.csv
- 注意：若需更准确的列表，可使用独立脚本 generate_stock_list_baostock.py（非 CLI 命令）。

### 2. 下载日线数据

bash复制下载

```text
python src/cli/claw_interface.py fetch_daily [--codes CODES] [--start START] [--end END] [--force]
```- --codes：股票代码，逗号分隔，默认使用全部主板股票
- --start：开始日期，默认 2005-01-01
- --end：结束日期，默认今天
- --force：强制重新下载（不检查本地已有）
- 示例：

bash复制下载

```text
python src/cli/claw_interface.py fetch_daily --start 2025-01-01 --end 2026-03-19
python src/cli/claw_interface.py fetch_daily --codes 000001,600036 --start 2025-01-01
```

### 3. 下载分钟线基础数据（1分钟或30分钟）

bash复制下载

```text
python src/cli/claw_interface.py fetch-minute-base --freq {1min,30min} --start START --end END [--codes CODES]
```- --freq：频率，1min 或 30min
- --start：开始日期（必填）
- --end：结束日期（必填）
- --codes：股票代码，逗号分隔，默认全部主板
- 示例：

bash复制下载

```text
python src/cli/claw_interface.py fetch-minute-base --freq 30min --start 2026-03-01 --end 2026-03-19
python src/cli/claw_interface.py fetch-minute-base --freq 1min --codes 000001 --start 2026-03-01 --end 2026-03-19
```

### 4. 下载指数数据

bash复制下载

```text
python src/cli/claw_interface.py fetch_index [--indices INDICES] [--start START] [--end END]
```- --indices：指数代码，逗号分隔，默认 000300.SH,000001.SH,399001.SZ
- --start：开始日期，默认 2005-01-01
- --end：结束日期，默认今天
- 示例：

bash复制下载

```text
python src/cli/claw_interface.py fetch_index
python src/cli/claw_interface.py fetch_index --indices 000300.SH,399006.SZ --start 2025-01-01
```

## 二、选股与策略指令

### 5. 执行多因子选股

bash复制下载

```text
python src/cli/claw_interface.py select-stocks [--date DATE] [--strategy-config CONFIG] [--factor-config CONFIG] [--output OUTPUT]
```- --date：选股日期，默认今天
- --strategy-config：策略配置，默认 configs/strategy.yaml
- --factor-config：因子配置，默认 configs/factors.yaml
- --output：输出 CSV 文件（可选）
- 示例：

bash复制下载

```text
python src/cli/claw_interface.py select-stocks --output stocks.csv
python src/cli/claw_interface.py select-stocks --date 2026-03-18
```

## 三、回测指令

### 6. 运行回测

bash复制下载

```text
python src/cli/claw_interface.py run-backtest [--config CONFIG] [--output-nav NAV_CSV] [--output-perf PERF_JSON]
```- --config：回测配置文件，默认 configs/backtest.yaml
- --output-nav：净值文件，默认 backtest_nav.csv
- --output-perf：绩效指标 JSON，默认 backtest_perf.json
- 示例：

bash复制下载

```text
python src/cli/claw_interface.py run-backtest
python src/cli/claw_interface.py run-backtest --config configs/backtest_ma.yaml
```

### 首次部署

bash复制下载

```text

python generate_stock_list_baostock.py

# 2. 下载指数数据

```### 每日例行

bash复制下载

```text

# 2. 执行选股
python src/cli/claw_interface.py select-stocks --output daily_stocks.csv
```### 策略验证

bash复制下载

```text

```以上指令已涵盖数据采集、选股、回测及辅助功能。如遇导入错误，请检查 VSCode 的 python.analysis.extraPaths 配置或确保在项目根目录运行命令。
