## 二、核心组件实现

### 2.1 因子配置加载器 (loader.py)

**功能**：
- 从 YAML 文件加载因子定义
- 提供因子查询接口
- 分析因子类型和依赖关系

**核心方法**：
```python
class FactorLoader:
    def load(self) -> Dict[str, Any]:  # 加载所有因子
    def get_factor(self, name) -> Dict:  # 获取单个因子
    def get_dependencies(self, name) -> List[str]:  # 获取依赖
    def get_factor_types(self) -> Dict:  # 按类型统计
```

**配置格式**：
```yaml
factors:
  ma5:
    type: technical
    function: moving_average
    params: {window: 5, field: close}
    inputs: [daily]
    description: 5日均线
```

### 2.2 因子计算器 (calculator.py)

**功能**：
- 实现各类因子计算函数
- 支持基础因子和复合因子计算
- 提供统一的计算接口

**支持的因子类型**：
1. **技术因子**：
   - `moving_average`: 移动平均线
   - `ratio`: 比值计算

2. **动量因子**：
   - `rsi`: 相对强弱指标

3. **成交量因子**：
   - `volume_ratio`: 成交量比值

4. **资金流向因子**：
   - `net_inflow_ratio`: 净流入占比

**核心方法**：
```python
class FactorCalculator:
    def calculate_moving_average(...)  # 均线
    def calculate_ratio(...)           # 比值
    def calculate_rsi(...)             # RSI
    def calculate_volume_ratio(...)    # 成交量比值
    def calculate_net_inflow_ratio(...)  # 净流入
    def calculate_all_factors(...)     # 批量计算
```

### 2.3 因子定义 (definitions.py)

**功能**：
- 定义因子的数据结构
- 提供类型检查和验证

**数据结构**：
```python
@dataclass
class FactorDefinition:
    name: str
    description: str
    category: str
    formula: Optional[str] = None
    parameters: Optional[List[str]] = None
```

### 2.4 因子函数库 (functions.py)

**功能**：
- 提供可复用的因子计算函数
- 支持自定义因子扩展

**实现函数**：
```python
def price_return(data) -> pd.Series  # 价格收益率
def volume_ratio(data) -> pd.Series  # 成交量比值
def ma_crossover(data) -> pd.Series  # 均线交叉
```

### 2.5 因子处理器 (processor.py)

**功能**：
- 批量处理因子计算
- 因子筛选和排序
- 因子标准化

**核心方法**：
```python
class FactorProcessor:
    def process(data, factor_names) -> pd.DataFrame  # 处理因子
    def filter_by_threshold(data, factor, threshold) -> pd.DataFrame  # 筛选
```

## 三、使用示例

### 3.1 基础使用流程

```python

factor_definitions = loader.load()

# 2. 准备数据
data = load_stock_data('600570.csv')

# 3. 计算因子
calculator = FactorCalculator()
results = calculator.calculate_all_factors(data, factor_definitions)

# 4. 分析结果
print(results['ma5'].tail())
print(results['rsi_14'].tail())
```

## 四、因子计算流程

### 4.1 数据准备阶段
1. 从数据源加载股票数据（OHLC、成交量、资金流向等）
2. 数据清洗和预处理
3. 数据对齐和合并

### 4.2 因子计算阶段
1. **加载配置**：从 YAML 文件读取因子定义
2. **基础因子计算**：计算不依赖其他因子的基础因子
3. **复合因子计算**：计算依赖其他因子的复合因子
4. **结果存储**：将计算结果保存到 DataFrame

### 4.3 因子分析阶段
1. **因子筛选**：根据阈值筛选符合条件的股票
2. **因子排序**：按因子值排序
3. **因子组合**：多因子加权合成
4. **可视化**：绘制因子分布和趋势

## 五、扩展指南

### 5.1 添加新因子

**步骤**：
1. 在 `configs/factors.yaml` 中添加新因子定义
2. 在 `calculator.py` 中实现计算函数
3. 在 `functions.py` 中添加可复用函数（可选）

**示例**：
```yaml
# 在 factors.yaml 中添加
ma30:
  type: technical
  function: moving_average
  params: {window: 30, field: close}
  inputs: [daily]
  description: 30日均线
```

```python
# 在 calculator.py 中添加
def calculate_moving_average_30(data, window=30, field='close'):
    return data[field].rolling(window=window).mean()
```
